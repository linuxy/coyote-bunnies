# coyote-bunny
A benchmark for [coyote-ecs](https://github.com/linuxy/coyote-ecs), a fast and simple zig native ECS.

```git clone --recursive https://github.com/linuxy/coyote-bunnies.git```

To build:
* zig build mimalloc
* zig build
